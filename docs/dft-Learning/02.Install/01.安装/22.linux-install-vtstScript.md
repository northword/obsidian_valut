# 配置 VTST 的脚本

## 下载

```bash
$ cd ~/app
$ wget http://theory.cm.utexas.edu/code/vtstscripts.tgz
```

## 解压缩

```bash
$ tar -xzvf vtstscripts.tgz 
$ rm vtstscripts.tgz
$ cd vtstscripts-（按Tab）
$ pwd
# 将返回值复制
```

## 加入 bashrc

```bash
vi ~/.bashrc
# 将上面pwd的返回值加入PATH
```
