Python 版本如下

```txt
Python 3.9.0 (tags/v3.9.0:9cf6752, Oct  5 2020, 15:34:40) [MSC v.1927 64 bit (AMD64)] on win32
```

Windows 版本如下：

```txt

版本号        20H2
安装日期      2020/11/2

序列号

```

搜索到一个类似问题的，是因为 `numpy` 与 `windows 20H2` 的关系（反正总是 Windows 的锅）。

使用如下将 `numpy` 降级即可。

```powershell
pip install numpy==1.19.3
```
